WASP2

Allele-Specific Analysis Pipeline

v1.3.0

A modern, Rust-accelerated pipeline for detecting allele-specific expression and chromatin accessibility. Corrects mapping bias with the proven WASP filtering strategy.

Quick Start

# Install from PyPI
pip install wasp2

# Or with conda

# Count alleles at variants
wasp2-count --bam reads.bam --vcf variants.vcf.gz
