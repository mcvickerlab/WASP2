Tutorials
=========

Learn WASP2 step-by-step with these progressive tutorials. Start with the concepts
if you're new to allele-specific analysis, or jump directly to the workflow that
matches your data type.

.. contents:: Tutorial Overview
   :local:
   :depth: 1

Learning Path
-------------

**New to allele-specific analysis?**
    Start with :doc:`concepts` to understand the biology and methods.

**Want to try WASP2 quickly?**
    Follow :doc:`/quickstart` for a 5-minute introduction.

**Ready for a complete workflow?**
    Work through :doc:`basic_workflow` to learn the full pipeline.

**Have specific data types?**
    - RNA-seq: :doc:`rnaseq_ase`
    - ATAC-seq: :doc:`atacseq_ase`
    - Single-cell: :doc:`single_cell`

Prerequisites
-------------

Before starting the tutorials, ensure you have:

1. **WASP2 installed**: See :doc:`/installation`
2. **Basic command-line knowledge**: Familiarity with bash/terminal
3. **Understanding of NGS data**: BAM and VCF file formats

Tutorial Format
---------------

Each tutorial follows a consistent structure:

- **Time estimate**: How long the tutorial takes
- **Prerequisites**: Required knowledge and data
- **Learning objectives**: What you'll accomplish
- **Step-by-step instructions**: Detailed commands with explanations
- **Expected output**: What success looks like
- **Troubleshooting**: Common issues and solutions
- **Next steps**: Where to go from here

Getting Help
------------

If you encounter issues:

1. Check the :doc:`/troubleshooting` guide
2. Review the :doc:`/faq`
3. Search existing `GitHub Issues <https://github.com/Jaureguy760/WASP2-exp/issues>`_
4. Open a new issue if your problem isn't addressed
